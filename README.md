# DriftQ-Starters
Starter templates for DriftQ: copy/paste production patterns for durable, replayable workflows.

## Templates

- **driftq-starter-fastapi-nextjs** => FastAPI API + Next.js UI + worker wired to DriftQ-Core via Docker Compose. Durable workflow pattern with SSE streaming and replay-friendly runs.
  - Use: Click “Use this template” → https://github.com/driftq-org/driftq-starter-fastapi-nextjs
